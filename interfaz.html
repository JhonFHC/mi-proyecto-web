Taller de Aplicaciones

Aprendizaje de Apache NetBeans, Maven, Diseño MVC y desarrollo web

Este taller está orientado a desarrollar habilidades prácticas en herramientas y patrones de diseño fundamentales para la ingeniería de sistemas y computación. A través de proyectos reales y metodologías ágiles, los estudiantes adquieren experiencia en el desarrollo de software profesional.

Objetivos del Taller

• Dominio de herramientas: Apache NetBeans, Maven, Git

• Patrones de diseño: MVC, principios SOLID

• Desarrollo web: HTML5, CSS3, JavaScript

• Bases de datos: MySQL, diseño relacional

Competencias Desarrolladas

• Programación: Java, POO, estructuras de datos

• Frontend: Interfaces responsivas, UX/UI

• Backend: Lógica de negocio, APIs

• Gestión: Control de versiones, documentación
