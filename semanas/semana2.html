Semana 2 - Métodos en Java

Aprende a estructurar tu código Java utilizando métodos para crear programas más organizados, mantenibles y reutilizables.

Conceptos de Métodos

Los métodos en Java son bloques de código que realizan una tarea específica y pueden ser reutilizados múltiples veces dentro de un programa.

Los métodos permiten dividir un programa complejo en partes más pequeñas y manejables, facilitando el mantenimiento y la depuración del código.

  • Métodos void: No retornan valor
  • Métodos con retorno: Devuelven un valor específico
  • Métodos con parámetros: Reciben datos de entrada
  • Métodos estáticos: Pertenecen a la clase, no a instancias
// Sintaxis básica de un método
[modificador] tipo_retorno nombreMetodo([parámetros]) {
    // Cuerpo del método
    return valor; // solo si no es void
}

// Ejemplos:
public void mostrarMensaje() {
    System.out.println("Hola Mundo");
}

public int sumar(int a, int b) {
    return a + b;
}

private double calcularPromedio(double[] notas) {
    double suma = 0;
    for(double nota : notas) {
        suma += nota;
    }
    return suma / notas.length;
}

Descripción del Problema

Durante la clase, se desarrollaron aplicaciones Java con interfaz gráfica para resolver problemas prácticos de cálculo de sueldos y ventas. El desafío consistía en implementar métodos especializados para cada funcionalidad, mantener una estructura modular y garantizar la validación de datos de entrada.

Solución Implementada

La solución incluyó la creación de métodos específicos para cada cálculo, separación de responsabilidades, implementación de validaciones de entrada y creación de interfaces de usuario intuitivas usando Java Swing.

Venta de Productos - Clase ImporteProducto

public class ImporteProducto extends JFrame {
    
    private void calcular() {
        double precio = Double.parseDouble(txtPrecio.getText());
        int cantidad = Integer.parseInt(txtCantidad.getText());
        
        double importe = precio * cantidad;
        double descuento = (cantidad >= 10) ? 
            0.20 * importe : 0.10 * importe;
        double total = importe - descuento;
        int obsequio = (total >= 200) ? 2 * cantidad : 0;
        
        mostrarResumen(importe, descuento, total, obsequio);
    }
    
    private void mostrarResumen(double importe, double descuento, 
                              double total, int obsequio) {
        txtSalida.setText("Resumen de compra:\\n");
        txtSalida.append(String.format("Importe: S/ %.2f\\n", importe));
        txtSalida.append(String.format("Descuento: S/ %.2f\\n", descuento));
        txtSalida.append(String.format("Total: S/ %.2f\\n", total));
        txtSalida.append("Obsequio: " + obsequio + " lapiceros");
    }
    
    private void limpiar() {
        txtPrecio.setText("");
        txtCantidad.setText("");
        txtSalida.setText("");
        txtPrecio.requestFocus();
    }
}

Métodos de Validación y Utilidades

// Método para validar entrada numérica
private boolean validarEntrada(String texto) {
    try {
        Double.parseDouble(texto);
        return true;
    } catch (NumberFormatException e) {
        JOptionPane.showMessageDialog(this, 
            "Error: Ingrese un valor numérico válido", 
            "Error de entrada", JOptionPane.ERROR_MESSAGE);
        return false;
    }
}

// Método para calcular descuentos progresivos
private double calcularDescuento(double importe, int cantidad) {
    if (cantidad >= 15) {
        return importe * 0.25;
    } else if (cantidad >= 10) {
        return importe * 0.20;
    } else {
        return importe * 0.10;
    }
}
