Conexión a Base de Datos

Implementación de la clase de conexión que maneja la comunicación segura entre la aplicación Java y la base de datos MySQL.

CConexion.java

package Clases;

import java.sql.Connection;
import java.sql.DriverManager;
import javax.swing.JOptionPane;

public class CConexion {
    Connection conectar;
    
    // Configuración de conexión
    String usuario = "admin";
    String contrasenia = "admin";
    String bd = "login";
    String ip = "localhost";
    String puerto = "3306";
    
    // Cadena de conexión con timezone UTC
    String cadena = "jdbc:mysql://localhost:3306/login?serverTimezone=UTC";
    
    public Connection estableceConexion() {
        try {
            // Cargar driver MySQL
            Class.forName("com.mysql.cj.jdbc.Driver");
            
            // Establecer conexión
            conectar = DriverManager.getConnection(cadena, usuario, contrasenia);
            
            // Comentado para producción
            // JOptionPane.showMessageDialog(null, "Se conectó correctamente a la Base de DATOS");
            
        } catch (Exception e) {
            JOptionPane.showMessageDialog(null, "Problemas en la conexión: " + e.toString());
        }
        return conectar;
    }
}
  • Driver MySQL: Uso de com.mysql.cj.jdbc.Driver para compatibilidad
  • Configuración Flexible: Parámetros separados para fácil mantenimiento
  • Manejo de Timezone: Configuración UTC para evitar problemas de hora
  • Gestión de Errores: Try-catch con mensajes informativos al usuario
  • Conexión Reutilizable: Método que puede ser llamado múltiples veces

Lógica de Autenticación

Implementación de la clase CLogin que valida credenciales contra la base de datos y maneja el flujo de autenticación.

Descripción del Módulo

La clase CLogin se encarga de validar las credenciales del usuario contra la base de datos MySQL, utilizando Prepared Statements para prevenir inyecciones SQL y manejando adecuadamente los resultados de la consulta.

Flujo de Autenticación

El sistema valida usuario y contraseña, consulta la base de datos, y según el resultado redirige al menú principal o muestra mensajes de error apropiados.

CLogin.java - Validación de Usuarios

package Clases;

import com.mycompany.loginn.FormMenuPrincipal;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import javax.swing.JOptionPane;
import javax.swing.JPasswordField;
import javax.swing.JTextField;

public class CLogin {
    public void validaUsuario(JTextField usuario, JPasswordField contrasenia) {
        try {
            ResultSet rs = null;
            PreparedStatement ps = null;
            
            // Crear conexión a la base de datos
            Clases.CConexion objetoConexion = new Clases.CConexion();
            
            // Consulta SQL con parámetros para prevenir SQL Injection
            String consulta = "SELECT * FROM Usuarios WHERE usuarios.ingresoUsuario = (?) AND Usuarios.ingresoContrasenia = (?);";
            ps = objetoConexion.estableceConexion().prepareStatement(consulta);
            
            // Obtener contraseña del campo
            String contra = String.valueOf(contrasenia.getPassword());
            
            // Establecer parámetros en la consulta
            ps.setString(1, usuario.getText());
            ps.setString(2, contra);
            
            // Ejecutar consulta
            rs = ps.executeQuery();
            
            // Verificar si se encontró el usuario
            if (rs.next()) {
                JOptionPane.showMessageDialog(null, "El Usuario es Correcto");
                
                // Redirigir al menú principal
                FormMenuPrincipal objetoMenu = new FormMenuPrincipal();
                objetoMenu.setVisible(true);
            } else {
                JOptionPane.showMessageDialog(null, "El Usuario es INCORRECTO, VUELVA A INTENTAR");
            }
            
        } catch (Exception e) {
            JOptionPane.showMessageDialog(null, "ERROR: " + e.toString());
        }
    }
}
